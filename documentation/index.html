popup.js

This JavaScript file provides some methods for showing popups. The following ID's and HTML-Classes are used by this JavaScript file:
  • #popup
  • #popup-button
  • #popup-button:hover
  • #popup-text
  • .yes-action
  • .ok-action
  • .no-action
  • .cancel-action


For correct usability you need the popup.css file.
